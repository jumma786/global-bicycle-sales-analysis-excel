# Global Bicycle Sales Analysis in Excel

## Project Overview
This project analyzes global bicycle sales data using Microsoft Excel.

The analysis includes:
- Data cleaning
- Pivot tables
- KPI calculations
- Revenue analysis
- Country-wise performance
- Interactive dashboard preparation

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Visualization

## Key Insights
- United States generated the highest revenue.
- Revenue trends vary across countries.
- Product categories contribute differently to profits.

## Dataset
The dataset contains sales transactions including:
- Revenue
- Profit
- Customer demographics
- Product categories
- Countries

- Data Cleaning
- Business Analysis
- Dashboarding
